# ドメインモデル（軽量スケッチ・最小限）

> bitz-ddd 未導入のため軽量ドメインスケッチとする。本ケースは完全ステートレスな
> 電卓CLIであり、永続化対象のエンティティが存在しないため最小限の記述に留める。

## 主要な概念（エンティティではなく値・振る舞い中心）

| 概念 | 種別 | 説明 |
|---|---|---|
| Expression | 値オブジェクト | ユーザーが入力する計算式（例: `1 + 2 * 3`）。実行間で保持されない |
| CalculationResult | 値オブジェクト | Expression の評価結果。標準出力に表示後は破棄される |

## 振る舞い

- 式を解析する（Expression → 構文木）
- 式を評価する（構文木 → CalculationResult）
- 結果を出力する（標準出力に表示、終了）

## 備考

- 「履歴保存なし・完全ステートレス」という要求（ユーザー指定）により、エンティティ間の
  永続的な関係やライフサイクルは存在しない。1回の実行が1回の計算で完結する。
