# ビジョン（Vision Board + PR-FAQ 圧力試験）

## Vision Board

- **For**: 日々のタスクを手軽に管理したい個人ユーザー
- **Who**: 付箋やメモアプリでタスク管理が煩雑になっている人
- **The** ToDo管理Webアプリ **is a** Webアプリケーション
- **That**: 一覧・追加・完了操作だけでタスクを管理できる
- **Unlike**: 高機能な多人数向けプロジェクト管理ツール
- **Our product**: MVC構成のシンプルな個人用ツールとして、学習コストゼロで使い始められる

## PR-FAQ 圧力試験（要旨）

- 見出し案: 「今日から使える、個人用ToDoアプリ」
- 想定FAQ: 「複数人で共有できますか？」→ 非対応（Non-Goal）。「モバイルアプリはありますか？」→ 対象外（Webのみ）。
- 崩壊クリティカル仮説: 「個人が最低限のCRUD（一覧・追加・完了）だけで満足するか」は `TBD`（未検証、ユーザーインタビュー未実施）。
