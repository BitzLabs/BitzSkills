# Mission & Vision

## Mission

スキル作者が SKILL.md を作るだけで終わらず、仕様検証、あり/なし比較テスト、評価、最適化、配布、運用後の改善まで一続きに実行できるようにする。

## Vision

Agent Skill の品質が主観的な文章レビューだけでなく、再現可能なテストと観察ログによって継続改善される状態を目指す。

## Discovery Gate
