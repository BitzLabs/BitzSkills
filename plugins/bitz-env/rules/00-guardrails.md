# bitz-env ガードレール（共通ルール）

このワークスペースには bitz-env プラグインのガードレールが適用されている。
（Antigravity はネイティブ rules/ として、Claude Code は SessionStart フックの
コンテキスト注入として、どちらのプラットフォームでも本文書が読み込まれる）

## 禁止（実行しない）

- `rm -rf` / `git push --force` / `git reset --hard` / `git clean -f` / `sudo`
  （PreToolUse フックでも機械的にブロックされる）
- デフォルトブランチへの直接コミット（変更時は必ずブランチを切る）
- 認証情報・トークン類の読み取り・出力

## 事前確認が必要（ユーザーの明示承認なしに実行しない）

- リポジトリ外への書き込み・上書き・削除

## 検証義務

- 委譲先・相談先（サブエージェント・外部エージェント）の「成功しました」という
  自己申告を信用せず、テスト・機械チェックを自分で再実行して確認する
- プロジェクトの AGENTS.md があればそちらの規約を優先する
