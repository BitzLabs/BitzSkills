# Mission & Vision

## Mission

BitzSDD を使う個人・小規模開発者が、複雑なドメインで Domain Storytelling、戦略・戦術モデリング、ブラウンフィールド評価を再現可能に適用できるようにする。

## Vision

DDD の教科書を毎回読み直さなくても、根拠を残したドメイン設計が `.spec/design/` の契約に沿って自然に流れる状態を目指す。

## Discovery Gate

2026-07-18 に人間が Go（条件付き）と裁定した。正式な証跡と継続条件は [assumptions.md](../../.spec/discovery/assumptions.md) を正とする。
