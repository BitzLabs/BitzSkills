## library 固有

- **クロスプラットフォーム/多環境マトリクス**:

  - TS: ESM/CJS・対象 TS バージョン・ブラウザ/Node で型と実行を検証

- **ダウンストリーム smoke**: 代表的な利用形態での最小結合確認
