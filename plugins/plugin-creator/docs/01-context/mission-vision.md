# Mission & Vision

## Mission

プラグイン作者が構造、コマンド、エージェント、フック、MCP、設定、スキル同梱の仕様差を調べ直さず、検証可能なプラグインを作れるようにする。

## Vision

複数プラットフォーム対応プラグインの設計・作成・検証が、一次情報に基づく再現可能なワークフローとして定着する状態を目指す。
