# Mission & Vision

## Mission

プロジェクト状況に応じたブランチ・worktree・Issue 駆動 PR の選択と、安全な復元手順をエージェントが一貫して案内できるようにする。

## Vision

単独開発から複数エージェント並列、公開チーム開発まで、毎回 Git フローを再設計せず事故の少ない運用を選べる状態を目指す。
