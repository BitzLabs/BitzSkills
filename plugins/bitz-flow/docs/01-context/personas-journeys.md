# Personas & Journeys

主要利用者は AI エージェントへ Git 操作を委ねる個人開発者。検証対象は SDD を採用せず bitz-flow 単体を使う開発者。

ペルソナの感情や外部利用者像は、実証されていない限り `[proto / 未検証]` として扱う。詳細は [personas.md](../../.spec/discovery/personas.md) を参照する。
