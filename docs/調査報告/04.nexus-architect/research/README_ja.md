# research/ — 背景調査ノート

ツールキットの設計決定の背後にあるソース調査資料：マイクロサービスアーキテクチャパターン、ScalarDBのユースケースとエディション、データモデリング、データベース調査、および関連トピック。ファイルはトピックごとに番号が付けられ、`_en.md` / `_ja.md` のペアとして維持されている。

これらのドキュメントは**参照資料のみ**である — ランタイム時にSkillやオーケストレーターによって読み込まれることはない。Skillが実際に消費する運用ガイダンスについては、`rules/` および `skills/common/references/` を参照すること。
