# スキル定義テンプレート集

本ドキュメントは、新規にエージェントスキルを作成する際のベーステンプレートを提供します。用途に応じて選択してください。

---

## 1. 最小構成テンプレート (シンプル)

複雑な補助スクリプトや長大なドキュメントを必要とせず、エージェントへの指示と行動ルールのみを拡張する場合に使用します。

### ディレクトリ構成
```
src/skills/<skill_name>/
└── SKILL.md
```

### `SKILL.md` テンプレート
```markdown
---
name: <skill_name>
description: <エージェントがこのスキルを読み込むトリガー条件を日本語で具体的に記述します。>
---

# <スキルタイトル>

*   このスキルは、ユーザーから「<トリガーフレーズや状況>」の指示があった場合に適用されます。

## 2. 実行手順
1.  **ステップ 1**: <具体的な作業手順>
2.  **ステップ 2**: <具体的な作業手順>
3.  **ステップ 3**: <具体的な作業手順>

## 3. 制約および行動ルール
*   **ルール 1**: <遵守しなければならない制約。例：コードコメントはすべて日本語にする、など>

```

---

## 2. 標準・複合構成テンプレート (推奨)

詳細なスキーマ、長大なマニュアル、サンプルコード、あるいは自動化用スクリプトを内包する、より高度で信頼性の高いスキルを作成する場合に使用します（段階的開示パターン）。

### ディレクトリ構成
```
src/skills/<skill_name>/
├── SKILL.md
├── references/
│   └── spec.md          # 詳細なAPI仕様やJSONスキーマ
├── examples/
│   └── good_sample.md   # インプットとアウトプットの理想的な実行例
└── scripts/
    └── helper.py        # 補助用の自動化プログラム
```

### `SKILL.md` テンプレート
```markdown
---
name: <skill_name>
description: <トリガー条件を具体的に記述します。>
---

# <複合スキルタイトル>

本スキルは、<目的>を行うための複合ガイドラインです。

## 1. 設計原則と段階的開示

*   詳細な仕様書・定義: [spec.md](file:///absolute/path/to/references/spec.md)
*   実行例とサンプル: [good_sample.md](file:///absolute/path/to/examples/good_sample.md)

## 2. 実行手順

3.  **検証と出力**: <手順>

## 3. 制約およびルール
*   **ルール 1**: メインの処理はスクリプトの出力を優先し、エラー発生時はログを出力すること。
```
